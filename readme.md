## how to run
1. Clone the repository
2. Run `docker-compose up` in the root folder
3. Access the front application at `http://localhost:4200`
4. Access the back application at `http://localhost:8080`

## project
This project is a **under-development** complete solution for any beauty business. It is created with the purpose of being a SaaS application with multiple business and multiple clients support.
